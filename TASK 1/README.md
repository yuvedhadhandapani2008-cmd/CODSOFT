# Rule-Based Chatbot using Python

## Description
This project implements a simple rule-based chatbot using Python.  
The chatbot interacts with users through the command line and responds
to predefined inputs using conditional statements.

## Features
- Responds to greetings such as hi, hello, and hey
- Answers basic questions like name, help, time and purpose
- Handles thank-you messages politely
- Exits gracefully when the user types "bye"
- Simple and beginner-friendly logic

## Technologies Used
- Python 3.x
- Visual Studio Code

## How to Run the Program
1. Make sure Python is installed on your system
2. Open the project folder in Visual Studio Code
3. Open the terminal
4. Run the program using:
        python "TASK 1 - CHATBOT.py"

## Sample Output
Bot: Hello! I am a simple rule-based chatbot.  
You: hi  
Bot: Hello! Nice to meet you.
